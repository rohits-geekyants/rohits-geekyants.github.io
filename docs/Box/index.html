Box

Generic component for the low level layout needs. It is similar to <div /> in HTML.

Implements

  • View from react-native
  • color, space, layout, flexbox & border from style-system

Example

(Note* Expo Snack goes here)

import React, { useState } from "react";
function Example() {
    // Declare a new state variable, which we'll call "count"  const [count, setCount] = useState(0);
    return (
        <div>
            <p>You clicked {count} times</p>
            <button onClick={() => setCount(count + 1)}>Click me</button>
        </div>
    );
}
<Box
    borderRadius={7}
    shadow={9}
    height={200}
    bg="indigo.8"
    children={
        <Text fontSize={30} color="white">
            Hello World!
        </Text>
    }
    justifyContent="center"
    alignItems="center"
/>;

Extra Props

  • shadow : Applies box shadow and accepts a number from 0 to 9 .
  • style : Applies user-defined styles and accepts a stylesheet.
  • children : Renders components as Box children. Accepts a JSX.Element or an array of JSX.Element.
