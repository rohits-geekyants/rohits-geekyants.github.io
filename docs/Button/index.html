Button

Generic component for the low level layout needs. It is similar to <button /> in HTML.

Implements

  • Ripple
  • color, space, layout & border from style-system

Example

<Button variant="promote" label="Login" shadow={4} mb={5} />
<Button
      variant="promote"
      label="Login"
      shadow={4}
      icon={{
          name: "md-checkmark-circle",
          type: "Ionicons",
          position: "right",
          style: { color: "red" },
      }}
/>
